Bamboo / Docs

Getting started with Bamboo is easy - you don't even have to be a web developer! Read through the comprehensive documentation below and you'll soon be a Bamboo expert.
Have an issue or problem that isn't explained below? Tell us.

First Steps

To start using Bamboo, copy the code snippet below and paste it right before the closing </head>.

<link rel="stylesheet" href="http://is.gd/bamboocss"><script type="text/javascript" src="http://is.gd/bamboojs"></script>

Here's how it looks when you put it into a blank HTML5 document:

<DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="http://is.gd/bamboocss">
<script type="text/javascript" src="http://is.gd/bamboojs"></script>
</head>
<body>
</body>
</html>

Getting Fancy Back to Top

Once you've pasted the Bamboo snippet into your website, you need to set it up so everything looks good. Here's how you do it.

Start by setting up your page so that Bamboo recognizes it and styles it just right. If you've already copied and pasted the blank template above, you just need to add a few things. First, add a <div class="wrapper"></div> and a <div class="main"></div> so your page looks like this:

<DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="http://is.gd/bamboocss">
<script type="text/javascript" src="http://is.gd/bamboojs"></script>
</head>
<body>
<header>
</header>
<nav>
<ul>
<li><a href="#">Home</a></li>
<li><a href="about.html">About</a></li>
<li><a href="new.html">News</a></li>
<li><a href="contact.html">Contact Us</a></li>
</ul>
</nav>
<div class="main">
This is where the main content goes.
</div>
</body>
</html>

Adding Page Elements Back to Top

Now that you've built a basic Bamboo web page, you need to add some important page elements, like a search bar, inputs, textareas, blockquotes, tables, buttons, and images.

Search Bars Back to Top

It's really easy to make a search bar look like the one below.

Just copy and paste this line of code into your website:

<input type="search" placeholder="Search">

You can also change around some of the attributes to change the outline color from the default blue to lightBlue, red, lightRed, green, lightGreen, orange, or lightOrange. Here's how it works.

Inputs Back to Top

Inputs are one-line text fields for users to enter data into. They are commonly used for usernames and passwords and are key elements for any website, so it's important to know how to use them. Bamboo input tags are the same as search tags, except you don't put the type="search". Customizing the color on an input is just as easy as customizing a search bar (you just choose one of the eight possible color choices). Here's how one looks:

Textareas Back to Top

Textareas are great when users need to enter in several sentences, and Bamboo textareas automatically expand. You can customize a textarea's outline color just like an input or search bar.

Just paste the code below for a textarea:

<textarea placeholder="Type on me!"></textarea>

Blockquotes Back to Top

Blockquotes are used to show a quote on your website. They're very simple to add but can't be customized (yet!).

Notice that the stiffest tree is most easily cracked, while the bamboo or willow survives by bending with the wind. - Bruce Lee

Just paste the code below for a blockquote:

<blockquote>The world is a wonderful place. - Eeyore</blockquote>

Images Back to Top

If you know some basic HTML, then you've probably worked with images before. Adding an image with Bamboo is as simple as <img alt="Example Image" src="example.png">. Here's an image with most of the possible class styles, which can be done with class="block bloom outline hover roundestHover"

Below are the different classes that can be applied to an image:

block (optional)

Positions the image on its own line, as opposed to the default inline styling.

bloom (optional)

Adds some shadow around the image on user hover.

outline (optional)

Adds a 2px outline around the image.

hover (optional)

Increases the opacity of the image when the user hovers over it.

roundHover (optional)

Slightly rounds the edges of the image on user hover.

rounderHover (optional)

Rounds the edges a little more than roundHover.

roundestHover (optional)

Rounds the edges a little more than rounderHover.

Progress Bars Back to Top

Progress bars are useful for showing the completion of a task, such as the loading of a website. Progress bars are one of the more complicated Bamboo elements to add to your website. Here's how you make a simple one:

<div class="meter"><span style="width: 70%"></span></div>

Animated progress bars are a little more difficult - here's how you make one:

<div class="meter animate"><span style="width: 70%"><span></span></span></div>
