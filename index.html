Welcome to Bamboo!

Bamboo is an amazingly simple HTML5 and CSS3 framework that provides the scaffolding you need to create a responsive, multi-platform web app. Check it out the official website at bamboo.jujubeweb.com.

Why Bamboo?

Always Updated

Never worry about using an "old" version of Bamboo. We never change the source CSS or JS files, so the Bamboo you linked to three decades ago will receive the newest updates.

120 characters of code

It's simple to install. Just copy and paste the 120 characters of code before the closing /head tag on all your website's pages and you are good to go!

Works on mobile devices

Bamboo is built to be responsive, so you don't have to worry about your websites looking ugly on your smartphone or tablet.

No license

That's the best part. Use, abuse, and reuse Bamboo like you made it! If you really like it, we ask that you pick up a Built with Bamboo badge and display it proudly on your website.
